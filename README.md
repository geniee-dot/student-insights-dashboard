# University Student Insights Dashboard

## Project Overview
This project analyzes student academic performance data and identifies at-risk students based on grade averages.

## Project Workflow
1. Data Cleaning (01_data_cleaning.ipynb)
2. Exploratory Data Analysis (02_eda.ipynb)
3. Interactive Dashboard (Streamlit)

## Key Features
- Final grade calculation
- At-risk student identification
- Study time vs performance analysis
- Absences vs grades visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run dashboard/app.py
